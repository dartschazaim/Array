## Реализованные функции

### 1. `void fillArray(int arr[], int size, int minVal, int maxVal)`
- **Назначение**: Заполнение массива случайными целыми числами
- **Параметры**:
  - `arr` - указатель на массив для заполнения
  - `size` - размер массива
  - `minVal` - минимальное значение для генерации
  - `maxVal` - максимальное значение для генерации
- **Возвращает**: void

### 2. `void printArray(int arr[], int size)`
- **Назначение**: Вывод массива целых чисел с индексами
- **Параметры**:
  - `arr` - указатель на массив для вывода
  - `size` - размер массива
- **Возвращает**: void

### 3. `int countInRange(int arr[], int size, int K, int L, int A, int B)`
- **Назначение**: Подсчет элементов массива, значения которых не входят в заданный диапазон значений
- **Параметры**:
  - `arr` - указатель на массив
  - `size` - размер массива
  - `K` - начальный индекс диапазона
  - `L` - конечный индекс диапазона
  - `A` - нижняя граница значений
  - `B` - верхняя граница значений
- **Возвращает**: количество элементов вне диапазона значений

### 4. `void fillArrayDouble(double arr[], int size, double minVal, double maxVal)`
- **Назначение**: Заполнение массива случайными вещественными числами
- **Параметры**:
  - `arr` - указатель на массив для заполнения
  - `size` - размер массива
  - `minVal` - минимальное значение для генерации
  - `maxVal` - максимальное значение для генерации
- **Возвращает**: void

### 5. `void printArrayDouble(double arr[], int size, const char* name)`
- **Назначение**: Вывод массива вещественных чисел с индексами
- **Параметры**:
  - `arr` - указатель на массив для вывода
  - `size` - размер массива
  - `name` - имя массива для вывода в заголовке
- **Возвращает**: void

### 6. `void processArrays(double a[], double b[], double c[], int n, int m, int k)`
- **Назначение**: Обработка трех массивов вещественных чисел
- **Параметры**:
  - `a` - указатель на первый массив (заменяется дробными частями)
  - `b` - указатель на второй массив (заменяется целыми частями)
  - `c` - указатель на третий массив (отрицательные заменяются модулями)
  - `n` - размер первого массива
  - `m` - размер второго массива
  - `k` - размер третьего массива
- **Возвращает**: void
- **Описание**: Для массива 'a' заменяет значения на их дробные части, для массива 'b' - на целые части, для массива 'c' заменяет отрицательные значения на их модули

### 7. `void selectionSort(int arr[], int n)`
- **Назначение**: Сортировка массива методом выбора
- **Параметры**:
  - `arr` - указатель на массив для сортировки
  - `n` - размер массива
- **Возвращает**: void

### 8. `void bubbleSort(int arr[], int n)`
- **Назначение**: Сортировка массива пузырьковым методом
- **Параметры**:
  - `arr` - указатель на массив для сортировки
  - `n` - размер массива
- **Возвращает**: void

### 9. `void cocktailSort(int arr[], int n)`
- **Назначение**: Сортировка массива шейкерным (коктейльным) методом
- **Параметры**:
  - `arr` - указатель на массив для сортировки
  - `n` - размер массива
- **Возвращает**: void

### 10. `void insertionSort(int arr[], int n)`
- **Назначение**: Сортировка массива методом вставок
- **Параметры**:
  - `arr` - указатель на массив для сортировки
  - `n` - размер массива
- **Возвращает**: void

### 11. `void generateRandomArray(int arr[], int n)`
- **Назначение**: Генерация массива случайных целых чисел
- **Параметры**:
  - `arr` - указатель на массив для заполнения
  - `n` - размер массива
- **Возвращает**: void

### 12. `void copyArray(int source[], int dest[], int n)`
- **Назначение**: Копирование содержимого одного массива в другой
- **Параметры**:
  - `source` - указатель на исходный массив
  - `dest` - указатель на целевой массив
  - `n` - размер массивов
- **Возвращает**: void

### 13. `void testSort(void (*sortFunc)(int[], int), int arr[], int n, const char* name)`
- **Назначение**: Тестирование алгоритма сортировки с измерением времени выполнения
- **Параметры**:
  - `sortFunc` - указатель на функцию сортировки
  - `arr` - указатель на массив для сортировки
  - `n` - размер массива
  - `name` - название алгоритма сортировки для вывода
- **Возвращает**: void
